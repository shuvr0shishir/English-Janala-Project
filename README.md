# 📘 English Janala

An interactive web application designed to make learning English easy and engaging for beginners. This platform helps users improve vocabulary with meanings, pronunciation, and real-time learning features.

🔗 Live Demo: https://english-janala-by-sk.netlify.app/  
🔗 GitHub Repo: https://github.com/shuvr0shishir/English-Janala-Project  

---

## 🚀 Features

- 🔐 **Authentication System**
  - Simple login validation
  - Protected content visibility
  - Logout functionality

- 📚 **Vocabulary Learning**
  - Learn words by lesson levels
  - Meaning & pronunciation for each word
  - Dynamic content loading from API

- 🔊 **Voice Pronunciation**
  - Click to hear correct pronunciation (Text-to-Speech)

- 🔍 **Search Functionality**
  - Search vocabulary instantly

- 🎯 **Lesson-Based Learning**
  - Filter words by lesson
  - Active lesson highlighting

- 📄 **Word Details Modal**
  - Example sentence
  - Synonyms
  - Detailed word information

- ❓ **FAQ Section**
  - Common JavaScript concepts explained

- ⚡ **Dynamic API Integration**
  - Fetch levels, words, and details from external API

- ⏳ **Loading Spinner**
  - Smooth UX while fetching data

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript (ES6)
- **Styling:** Tailwind CSS / DaisyUI
- **API:** Programming Hero Open API
- **Voice API:** Web Speech API

---

## 📡 API Endpoints

- Get All Levels  
  `https://openapi.programming-hero.com/api/levels/all`

- Get Words by Level  
  `https://openapi.programming-hero.com/api/level/{id}`

- Get Word Details  
  `https://openapi.programming-hero.com/api/word/{id}`
